Qui sommes-nous

Nous sommes deux étudiants ingénieurs à l'ISEN de Nantes. Nous présentons sur ce site web un projet de Master 1 :

Reconstruction via apprentissage profond pour des données fonctionnelles partielles.

Le but de ce projet est de reconstruire des données fonctionnelles partielles à l’aide de deep learning. Nous reconstruisons les parties manquantes (ou non observées) de fonctions à l’aide de leurs parties observées. Pour cela, nous utilisons un réseau de neurones du type RNN préalablement entraîné sur des jeux de données simulées. .

A propos


Ce projet a été proposé par Matthieu Saumard enseignant chercheur en mathématiques à l’ISEN de Nantes. Plusieurs sous-objectifs ont été réalisés pendant ces trois mois de projets. Des livrables finaux sont disponibles en annexe du rapport. Le projet a pour objectif de reconstruire des données fonctionnelles à l’aide de deep learning. Les données sont simulées partiellement afin de reconstruire des données fonctionnelles complètes. La reconstruction est possible à l’aide d’un réseau de neurones préalablement entraîné sur des jeux de données simulées. Ce projet est une jonction entre le domaine de l’analyse de données fonctionnelles, celui des techniques neuronales et le développement web avec l'application.
